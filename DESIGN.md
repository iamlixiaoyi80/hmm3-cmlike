# 英雄无敌：王国征服 - 游戏设计文档

> **类型：** CMlike（类 Coin Master / Monopoly Go）
> **题材：** 英雄无敌3（Heroes of Might and Magic 3）
> **设计日期：** 2026-03-07
> **设计者：** Claude Code + 重生小爪
> **版本：** v1.2

---

## 零、目标受众

### 核心用户
**SLG + HMM3 用户**
- 年龄：28-45 岁
- 特点：有策略游戏经验，熟悉 H3 世界观
- 需求：怀旧情怀 + 轻度策略 + 收集养成
- 付费能力：中高（有稳定收入）
- 游戏时间：碎片化（通勤、休息间隙）

### 次核心用户
**西幻用户 + 卡牌用户**
- 年龄：22-38 岁
- 特点：喜欢奇幻题材，有卡牌收集习惯
- 需求：英雄收集 + 卡牌养成 + 视觉满足
- 付费能力：中等
- 游戏时间：较碎片化

### 泛用户
**休闲用户 + 模拟经营用户**
- 年龄：18-35 岁
- 特点：休闲游戏经验，喜欢建造/经营
- 需求：轻松上手 + 建造反馈 + 社交互动
- 付费能力：中低
- 游戏时间：高度碎片化

### 用户优先级矩阵

| 用户类型 | 优先级 | 设计侧重 |
|----------|--------|----------|
| SLG + HMM3 用户 | 🔴 最高 | H3 元素还原、策略深度 |
| 西幻 + 卡牌用户 | 🟡 高 | 英雄设计、卡牌系统 |
| 休闲 + 经营用户 | 🟢 中 | 简化操作、建造反馈 |

## 一、核心循环

### 主循环
```
掷骰子（可操作） → 移动/收集资源 → 建设城堡 → 解锁新地图 → 收集英雄卡牌 → 强化战力 → 挑战更强玩家/副本
```

### 次级循环
1. **战斗循环：** 攻击其他王国 → 掠夺资源 → 被复仇 → 防御加固
2. **收集循环：** 开宝箱 → 获得英雄卡 → 合成升级 → 解锁技能
3. **社交循环：** 邀请好友 → 互助建设 → 组队副本 → 公会战

## 二、核心玩法

### 1. 骰子系统（B+E方案升级版）

#### 骰子操作机制
- **点击掷出** → 骰子开始旋转
- **旋转过程中** → 骰子快速滚动显示不同数字（1-6）
- **玩家点击"停止"** → 骰子减速停下，定格在当前数字
- **技能上限** → 玩家可以通过练习提高"停在想要数字"的成功率，但仍有随机性

#### 视觉反馈（Juice）
| 时刻 | 反馈 |
|------|------|
| 骰子掷出 | 骰子从手中飞出，带有拖尾特效 |
| 骰子旋转 | 慢动作特写，数字发光闪烁，音效"叮叮叮" |
| 玩家点击停止 | 屏幕轻微震动，"锁定！"文字弹出 |
| 骰子停下 | 数字放大，光芒爆发，对应事件图标飞入 |
| 获得资源 | 金币/资源从骰子中心飞出，计数器快速滚动 |
| 连续获得 | 连击提示（Double! Triple!），奖励加成 |

#### 音效设计
- 骰子旋转：节奏加快的"咔咔咔"声
- 点击停止：清脆的"叮！"锁定音
- 获得资源：金币掉落声，计数声
- 特殊事件：独特的音效（宝箱开启、战斗号角等）

#### 骰子类型
| 骰子类型 | 效果 | 视觉差异 |
|----------|------|----------|
| 普通骰子 | 基础移动 | 白色骰子 |
| 黄金骰子 | 双倍收益 | 金色骰子，旋转时有金色粒子 |
| 攻击骰子 | 必定触发攻击事件 | 红色骰子，旋转时有火焰特效 |
| 盾牌骰子 | 额外防御加成 | 蓝色骰子，旋转时有护盾光环 |

**获取方式：**
- 自动恢复：每 10 分钟 1 个（从 30 分钟优化）
- 完成任务：任务奖励体力
- 商店购买
- 上限：50 个（可升级扩展）

#### 骰子用完后的解决方案

**核心循环升级：**
```
消耗骰子 → 获得物资 → 提升实力 → 完成任务 → 获得体力奖励 → 继续消耗骰子
     ↑                                                        ↓
     ←←←←←←←←←←←← 体力自然恢复 ←←←←←←←←←←←←←←←←←←←←←←←←←←←←←
```

**体力系统设计：**

| 体力来源 | 获取量 | 说明 |
|----------|--------|------|
| 自然恢复 | 1 骰子/10分钟 | 离线也恢复 |
| 任务奖励 | 3-10 骰子 | 根据任务难度 |
| 成就奖励 | 5-20 骰子 | 一次性奖励 |
| 商店购买 | 不限 | 付费 |

**任务系统设计：**

| 任务类型 | 示例 | 奖励 |
|----------|------|------|
| 每日任务 | 采集 1000 金币 | 3 骰子 |
| 每日任务 | 升级 1 个建筑 | 3 骰子 |
| 每日任务 | 攻击 3 个玩家 | 5 骰子 |
| 周任务 | 收集 5 张英雄卡 | 10 骰子 |
| 成就 | 累计获得 10000 金币 | 5 骰子 |
| 成就 | 解锁第 2 个城堡 | 20 骰子 |

**效果：**
- 活跃玩家每天可通过任务获得 20-30 额外骰子
- 非付费玩家也有充足的游戏时间
- 引导玩家完成核心玩法（采集、建造、攻击）

```

│         🎁 宝箱发现！           │
```

- 新手期 → 选金币（建设急需）

### 3. 城堡建设（MVP简化版）

#### 种族选择（游戏开始一次性选择）

| 城堡 | 种族 | 特色 | 适合玩家 |
|------|------|------|----------|
| 🏰 骑士堡 | 人族 | 均衡发展，天使压阵 | 新手/均衡型 |
| 🌲 森林堡 | 精灵 | 资源加成20%，龙族后期 | 经济型/后期玩家 |
| 💀 墓园 | 亡灵 | 复活机制，数量压制 | 策略型/喜欢堆量 |

**后续版本解锁：** 地狱堡、据点、塔楼、要塞、元素堡

#### 建筑类型

**1. 资源建筑**
- 金矿、木材厂、矿石场、水晶矿等
- 被动产出资源

**2. 生物建筑（每种族7个）**
- 1-7级生物巢穴
- 可升级（1-100级）
- 升级提升待招募池增长速度

**3. 特殊建筑**
- 酒馆：英雄招募
- 魔法塔：魔法研究
- 防御工事：城堡防御

**建造反馈：**
- 建造开始：地基动画，工人出现
- 建造完成：建筑升起动画，烟花庆祝
- 升级完成：建筑变化，星级提升

### 4. 生物系统（H3完整还原）

#### 生物分级（每种族7级）

| 级别 | 骑士族 | 精灵族 | 亡灵族 | 招募价格 |
|------|--------|--------|--------|----------|
| 1级 | 枪兵 → 长枪兵 | 半人马 → 半人马首领 | 骷髅 → 骷髅勇士 | 60金/个 |
| 2级 | 弓箭手 → 神射手 | 矮人 → 战斗矮人 | 僵尸 → 僵尸领主 | 100金/个 |
| 3级 | 狮鹫 → 皇家狮鹫 | 精灵 → 大精灵 | 幽灵 → 怨灵 | 200金/个 |
| 4级 | 剑士 → 十字军 | 飞马 → 银飞马 | 吸血鬼 → 吸血鬼王 | 300金/个 |
| 5级 | 僧侣 → 狂热者 | 树精 → 枯木卫士 | 尸巫 → 尸巫王 | 500金/个 |
| 6级 | 骑兵 → 冠军骑士 | 独角兽 → 战争独角兽 | 死骑 → 地狱骑士 | 1000金/个 |
| 7级 | 天使 → 大天使 | 绿龙 → 金龙 | 骨龙 → 幽灵龙 | 3000金/个 |

#### 生物数量机制

- **显示格式：** 天使 ×5（带数量，非单卡）
- **生物上限：** 无上限

#### 生物获取途径

**途径1：投骰子生物地格**
- 踩到生物地格 → 实时获得
- 获得概率与对应生物建筑等级挂钩
- 高级生物极低概率，数量稀少

**途径2：生物建筑招募**
- 待招募池按时增加（如 15个/小时）
- 待招募池 ≥1 时可花费资源招募
- 招募后立刻获得

#### 待招募池增长速度

| 生物等级 | 基础增长 | 投骰子概率 |
|----------|----------|------------|
| 1级 | 15个/小时 | 30% |
| 2级 | 12个/小时 | 25% |
| 3级 | 8个/小时 | 20% |
| 4级 | 5个/小时 | 12% |
| 5级 | 3个/小时 | 7% |
| 6级 | 1个/小时 | 4% |
| 7级 | 0.5个/小时 | 2% |

**建筑等级加成：**
```
实际增长 = 基础增长 × (1 + 建筑等级 × 1%)
```

**示例：**
```
天使建筑 Lv5
→ 待招募池 +0.5个/小时 × (1 + 5 × 1%) = 0.525个/小时
→ 约2小时后池中有1个天使
→ 花费3000金 → 立刻获得 天使 ×1
```

#### 生物升级形态

- **解锁条件：** 生物建筑达到10级（可配置）
- **升级效果：** 枪兵 → 长枪兵（属性提升）
- **H3还原：** 建筑升级解锁高级形态

**卡牌品质：** 普通 < 稀有 < 史诗 < 传说

**MVP阶段英雄（每种族3位，共9位）：**

| 种族 | 英雄 | 技能 |

| 骑士族 | 克里斯丁 | 战斗狂热：攻击伤害+8% |
| 骑士族 | 罗德哈特 | 骑兵精通：骑兵攻击+15% |

| 精灵族 | 瑟琳娜 | 自然守护：精灵族防御+10% |

- 每星解锁1个技能槽

- 卡牌翻转动画
- 英雄登场立绘
- 稀有度对应光芒颜色

### 7. 战斗系统

**攻击流程：**
1. 掷到攻击事件
2. 选择目标
3. 投掷战斗骰子
4. 战力计算：英雄战力 + 建筑加成 + 骰子加成

**结果判定：**
- 大胜：掠夺200%资源
- 小胜：掠夺100%资源
- 平局：掠夺50%资源
- 失败：无收获，对方复仇值+1

**防御机制：**
- 城堡耐久度（基础1000，可升级）
- 防御陷阱（自动反击）
- 护盾系统（免疫攻击次数）

## 四、社交裂变

## 五、差异化亮点

### 1. 深度策略玩法
- **英雄技能：** 主动技能 + 被动技能 + 组合技能
- **阵型搭配：** 最多3位英雄，职业搭配，属性克制

### 2. RPG成长系统
- 英雄培养：等级、装备、天赋树
- 城堡专精：战争/贸易/防御/魔法

### 3. 副本探索
- 单人副本：战役模式、Boss挑战、无尽塔
- 团队副本：3-5人协作

### 4. PVP竞技场
- 赛季制：每2周一赛季
- 段位系统：青铜→白银→黄金→钻石→大师→传奇

### 5. 赛季活动
- 赛季主题：埃拉西亚的混乱、精灵之歌、地狱之门等
- 赛季奖励：限定英雄、限定皮肤、专属称号
